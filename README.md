# Micro-Ubiquitous-Accounting-Notary
 (MUAN) Programming Language is an expression-only, interpreted  programming language designed to run on an embedded system without any hardware floating-point  support
Core Topics: Floating Point, Bitwise Operators, C Programming Design (Helper Functions)
implemented functions to create the MicroSF 13-bit floating point library that MUAN will be 
using. Completed six functions in src/microsf.c
Completed this code, along with any number of helper functions that you would like to use, to implement 
these six functions.  In this project, There are custom microsf_s type variables.  These 
custom types are 16-bit unsigned short in memory.  Within these 16-bits, encoding custom 13-bit floating point value.
